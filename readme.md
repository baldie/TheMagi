**The Magi**

A personal, omnipresent Artificial Intelligence system, built to serve a human as an omi-present board of directors. Built on Windows using WSL.

* Caspar - Mistral
* Melchior - Gemma
* Balthazar - Llama2

To run, clone the repo, launch a terminal and run:

```
install-magi.bat
```

and then

```
start-magi.bat
```
