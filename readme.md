**The Magi**

A personal, omnipresent Artificial Intelligence system, built to serve a human as an omniscient board of directors. Built on windows using WSL

* Casper - Mistral
* Melchior - Gemma
* Balthazar - Llama2

To run, clone the repo, launch a terminal and run:

```
start-dev.bat
```
